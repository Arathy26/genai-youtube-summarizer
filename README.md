# GenAI YouTube Summarizer

AI-powered YouTube transcript summarizer built using LangChain, Groq, ChromaDB, and Streamlit.

---

## Features

- Extract YouTube transcripts
- AI-powered video summarization
- Ask questions about videos
- Semantic search using RAG
- Streamlit frontend

---

## Technologies Used

- Python
- Streamlit
- LangChain
- Groq
- ChromaDB
- HuggingFace Embeddings

---

## Run Locally

### Clone Repository

git clone https://github.com/Arathy26/genai-youtube-summarizer.git

### Install Requirements

pip install -r requirements.txt

### Run App

streamlit run app.py
